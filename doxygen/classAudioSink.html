Detailed Description

An output stream for an Audio file.

An AudioSink consists of a PortAudio output stream and a buffer that stores decoded samples from the Audio object. While active, the AudioSink periodically transfers samples from its buffer to PortAudio in a separate thread.

Transfers a range of sample bytes into the AudioSink.

The range may be empty, but must be valid.

  • Precondition: start <= end, start and end point to a valid contiguous block of sample bytes.
